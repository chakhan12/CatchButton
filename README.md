## 주요 기능

- 버튼이 마우스를 피해 랜덤 위치로 이동
- 버튼 도망 시 효과음 출력
- 버튼 클릭 성공 시 축하 메시지 출력
- 버튼 위치 좌표를 창 제목에 표시
