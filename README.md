# Catch Button Game 🎮

## 프로젝트 소개
C# Windows Forms를 사용하여 만든 숨바꼭질 버튼 게임입니다.  
마우스를 버튼 위에 올리면 버튼이 랜덤 위치로 도망가고,  
버튼을 클릭하면 성공 메시지와 효과음이 출력됩니다.

사용자는 도망가는 버튼을 잡는 간단한 게임을 통해  
이벤트 처리(Event Handling)와 폼 속성(Property)을 활용하는 방법을 학습할 수 있습니다.

---

## 주요 기능

- 버튼이 마우스를 피해 랜덤 위치로 이동
- 버튼 도망 시 효과음 출력
- 버튼 클릭 성공 시 축하 메시지 출력
- 버튼 위치 좌표를 창 제목에 표시

## 개발 단계

### 1단계 : 버튼 도망 기능 구현
마우스가 버튼 위에 올라가면 버튼이 랜덤 위치로 이동하도록 구현했습니다.  
이를 위해 `MouseEnter` 이벤트와 `Random` 클래스를 사용하여 버튼의 위치를 변경했습니다.

<img width="792" height="472" alt="22" src="https://github.com/user-attachments/assets/42b694f0-c8ab-4c6c-8c61-7a5998982b2c" />

---

### 2단계 : 시각적 / 청각적 피드백 추가
버튼이 도망갈 때 효과음을 출력하고,  
버튼 클릭에 성공했을 때 축하 메시지 박스를 표시하도록 구현했습니다.

- 도망갈 때 : Beep 효과음 출력  
- 클릭 성공 : 축하 메시지 출력

이를 통해 사용자가 게임 진행 상황을 쉽게 인식할 수 있도록 했습니다.
